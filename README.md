
# Java Exercises 

Java programming exercises completed as part of my CS studies at PJATK, Warsaw.

## Exercises

**Task1.java — Sorting three numbers**
Takes 3 integers as input and sorts them in ascending order using a swap algorithm.

**Task2.java — Longest consecutive sequence**
Reads a sequence of numbers and finds the longest run of consecutive identical values.

**Task3.java — Asterisk bar chart**
Takes 3 non-negative integers and prints a vertical bar chart made of asterisks.

**Arr2DMisc.java — 2D array operations**
Implements inner matrix extraction, matrix transposition, and right-half extraction on 2D arrays.

**StringMisc.java — String manipulation**
Implements name normalisation, initials formatting, and character translation functions.

**MergingF.java — Array merging**
Merges two arrays into one sorted array.

**DatesSimple.java — Bit manipulation**
Encodes and decodes date ranges using bitwise operations.

Java
